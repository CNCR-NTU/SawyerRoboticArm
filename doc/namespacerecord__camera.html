def record_camera.main ( )

Main function to record the video.


Initialize the ROS node and read the arguments given when launching the script The arguments can include the choice of the camera, the type of image displayed (raw, corrected or with edge detection), gain and exposure and the title of the video 

Camera Display Example

Cognex Hand Camera Ranges
    - exposure: [0.01-100]
    - gain: [0-255]
Head Camera Ranges:
    - exposure: [0-100], -1 for auto-exposure
    - gain: [0-79], -1 for auto-gain
